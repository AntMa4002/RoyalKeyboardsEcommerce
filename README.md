# RoyalKeyboardsEcommerce

## Abstract
An Ecommerce website that utilizes the MERN Stack

### Technologies Used
TypeScript, MongoDB, Mongoose, Twingate, AWS(EC2), Express, React, Ts-node, Nodemon, Bcrypt, Cors, Jsonwebtoken, HTML, CSS, Bootstrap
